# Lolconfig

lolconfig is a lightweight way to configure your ruby app.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'lolconfig'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install lolconfig
